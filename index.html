Timestampz

Timestampz is an student-tracking web application that will be used by Citizen Schools in East Palo Alto, CA in the 2012-13 school year. Individual teachers can check attendance and assignments in a quick and organized manner, while Administrators can get a high level overview of each classroom's statistics updating in real-time from the dashboard. The app also allows the Excel imports for student rosters, as well as exports of data.

It is currently in production, so it is inaccessible through Heroku. However, you can fork the repository here and run a local instance to see all the features.

Collaborators: Felix Tsai, Brian Parks, and Austin Wang
